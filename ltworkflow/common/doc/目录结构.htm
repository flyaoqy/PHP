文件目录(蓝色为文件夹) 说明
workflow.php 　 　 　 工作流入口文件
service 　 　 　 工作流核心处理文件夹
TaskService.php 　 　 流程任务处理类，通过、驳回、会签等等
InfoService.php 　 　 信息服务类，查询流程信息
CommandContext.php 　 　 命令上下文，包含所有全局参数
ConfigContext.php 　 　 配置上下文，记录xml配置信息
command 　 　 命令文件夹，包含所有流程动作
Command.php 　 命令接口
InitCommand.php 　 流程初始化
TaskCommand.php 　 流程发起
DoFreeRouteCommand.php 　 会签确认
DoOtherRouteCommand.php 　 循环会签确认
FreeRouteCommand.php 　 会签发起
OtherRouteCommand.php 　 循环会签发起
RecycleCommand.php 　 收回
RejectCommand.php 　 驳回
FreeTaskCommand.php 　 自由流程发起
impl 　 　 动作具体实现文件夹
InitCommandImpl.php 　 流程初始化实现
TaskCommandImpl.php 　 流程发起实现
RejectCommandImpl.php 　 驳回、收回实现
FreeRouteCommandImpl.php 　 会签相关实现
FreeTaskCommandImpl.php 　 自由流程实现
behavior 　 流程出口状态文件夹
Behavior.php 状态接口
BehaviorFactory.php 流程出口判断
NormalBehavior.php 正常通过到下一节点
OrgBehavior.php 组织结构审批
AutoBehavior.php 自动通过到下一节点
EndBehavior.php 审批结束或子流程结束
SubBehavior.php 子流程审批
ParallelBehavior.php 并行流程审批
FreeBehavior.php 自由流程审批
config 　 　 配置文件夹
config.php 　 用户配置，人员部门信息、邮件通知等
db 　 　 数据库操作文件夹
TaskDB.php 　 流程动作
InfoDB.php 　 流程信息读取
RejectDB.php 　 驳回
FreeRouteDB.php 　 会签
FreeTaskDB.php 　 自由流程
common 　 数据库操作封装
DBcommon.php 数据库操作封装
util 　 　 工具文件夹
Common.php 　 全局方法
XmlUtil.php 　 xml相关操作
ClassUtil.php 　 反射类操作
ErrorBox.php 　 错误处理
inheritance 　 　 继承类或接口
WorkflowBase.php 　 自定义脚本继承此类
Restrict.php 　 流程校验类模板
Results.php 　 条件类模板
Roles.php 　 角色类模板
Func.php 　 流程后处理模板
BeforeAction.php 　 下一节点前处理模板
InterfaceCounterSign.php 　 会签接口
InterfaceInit.php 　 用户初始化接口
InterfaceReject.php 　 驳回接口
InterfaceNotice.php 　 通知信息接口
InterfaceUserDB.php 　 人员信息接口，取人员、部门等
AbstractOrgRule.php 　 组织结构审批，用户自定义文件需继承此类
custom 　 　 　 用户自定义脚本文件夹（可修改名称）
common 　 　 用户工具
database 　 　 用户数据库操作，实现InterfaceUserDB.php
email 　 　 邮件通知，实现InterfaceNotice.php
freeconfig 　 　 自由流程
function 　 　 流程后处理，继承Func.php
驳回，实现InterfaceReject.php
会签，实现InterfaceCounterSign.php
init 　 　 流程初始化，实现InterfaceInit.php
organization 　 　 组织结构审批，继承AbstractOrgRule.php
restrict 　 　 流程前校验，继承Restrict.php
下一节点前处理，继承BeforeAction.php
results 　 　 条件判断，继承Results.php
roles 　 　 角色判断，继承Roles.php
