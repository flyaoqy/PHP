7、  审批配置表t_wf_steps_condition

描述：记录图形化页面配置的审批条件，审批人信息。同一环节的多条记录按顺序判断，有一个表达式返回true则返回true。

字段

类型

描述

id

int

主键id

wf_uid

varchar(50)

流程id

type

varchar(50)

类型：

Condition审批条件配置

Role审批人配置

step_id

varchar(50)

当前审批环节

next_step_id

varchar(50)

下一审批环节（type=role时为空）

expression

varchar(2000)

条件判断表达式，1 表示返回true

group_uid

varchar(50)

审批人配置（type=Condition时为空）

sort

Int

条件判断顺序
